Привет! В днном репрозитории 2 кода в пайтон: сборщики данных о стоимости яблок и груш с сайта agrobazar.ru

Он создает csv таблицу с стоимостью продажи яблок в каждую дату когда они продавались на сайте.
Дни недели для удобства отделены от даты.

В репрозитории есть 2 файла результата работы кода

Для работы необходимы библиотеки selenium и pandas
Так как код собирает данные с Гугл Хрома, необходимо бросить в папку с кодом вебдрайвер хрома
Ссылка для скачивания вебдрайвера: https://drive.google.com/open?id=1809ER0hvH56JNOrhpUwrobrnH-Xz3aTN

Язык: Python 3.7
